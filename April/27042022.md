## Learning Points

- TextInput is a component for inputting text into the app via a keyboard
- Some examples of TextInput props includes:
- placeholder: string that will be rendered before text input has been entered.
- secureTextEntry: default value is false. If set totrue, the text input obscures the text entered so that sensitive text stay secure.
- autoCapitalize: Tells TextInput to automatically capitalize certain characters.
- autoCorrect: If false, disables auto-correct. The default value is true.
- onChangeText: Callback that is called when the text input's text changes. Changed text is passed as a single string argument to the callback handler.
- value: The value to show for the text input. TextInput is a controlled component, which means the native value will be forced to match this value prop if provided.
